Angular Directive for LastFM
====================

Usage:
---------------------
	<ul class="lastfm" 
		lastfm="USERNAME" 
		period="PERIODS -- SEE LASTFM DOCS" 
		limit="LIMIT" 
		data-key="API KEY">
	</ul>
    
Example:
---------------------
	<ul class="lastfm" 
		lastfm="countymojo" 
		period="3month" 
		limit="10" 
		data-key="12345678901234567890">
	</ul>
